## Containerize Infrastructure Dependencies

La aplicación se ejecutará localmente con PHP, pero las dependencias de infraestructura (inicialmente MariaDB) se ejecutarán en contenedores Docker.

¿Por qué?

Mantiene limpio el sistema operativo.
Facilita la configuración del entorno.
Hace que cualquier desarrollador pueda levantar la base de datos con un solo comando.
Es el primer paso hacia una futura contenerización completa.
